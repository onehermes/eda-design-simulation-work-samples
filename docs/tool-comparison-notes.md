# Tool Comparison Notes

| Tool | Purpose | Typical Use Here | Notes |
| --- | --- | --- | --- |
| KiCad | Schematic capture and PCB layout | Review schematic flow, ERC/DRC, and footprint assignment | Strong open-source PCB design suite |
| LibrePCB | EDA tool for schematic and board design | Compare workflow organization and library handling | Useful reference for structured hardware design workflows |
| Ngspice | SPICE circuit simulator | Run netlist-based analog simulations | Good for checking frequency and transient behavior |
| Qucs-S | GUI front end for simulation workflows | Document simulation setup and result interpretation | Helpful for visual circuit exploration |

## Scope Note

This comparison describes the tools at a practical level for portfolio work. It does not claim deep specialist mastery of every feature.
